# 麻里

- 初登場: 第1話冒頭。
- 元の世界では30歳の料理研究家。
- 料理研究家としての限界を感じ、YouTube動画で生活していた。
- 動画テーマは「なろう系小説の世界の道具を使って、料理・保存食をつくる」。
- 干し肉、干し魚、固焼きパンなどの保存食作りに詳しい。
- 前世のYouTube動画で、なろう系小説の世界にありそうな道具を使い、保存食がどれくらい保つかの経過確認も実際に行っていた。そのため、干し肉や固焼きパンの保存中に起きる見た目、匂い、硬さ、湿り戻り、カビなどの変化について実地の知識がある。
- 保存食をスープなどに利用し、キャンプでアウトドア料理も作っていた。
- 自作の皮袋、小さな木箱、小さめの樽など、保存容器にもこだわる。
- 異世界で初めて得た獲物の肉を使い、干し肉の仕込みと胃袋詰めを作った。
- 家にある粗い穀粉、水、塩、畑のハーブを使い、保存食用の固焼きパンを作った。
- 燻製に使う枝は、樹脂臭の強いものを避け、乾いた広葉樹の枝を選ぶ。
- ブランが狩った鹿に似た獣の皮を簡易的になめし、家にあった裁縫道具で保存食を運ぶための皮袋を作った。
- 街へ行くため、皮袋に干し肉、固焼きパン、少量の塩、畑のハーブ、水を入れた皮袋、布、包丁を詰めた。
- 街では「森の外れの小屋に住む保存食作りの女で、干し肉の売り先を探している」と説明する方針を立てた。
- 門番には「マリ」と名乗り、保存食作りの女として干し肉の売り先を探していると説明した。
- 流浪市で干し肉を売り切ったあと、岩塩1袋、銀貨11枚、銅貨10枚を持って森の家へ帰った。
- 家に戻ると、小さなツボに銀貨11枚をしまい、岩塩1袋を貯蔵庫へしまった。
- 初回の干し肉販売の成功を受け、次は干し肉を長期保存できる品質まで完成させようと決めた。
- 流浪市から帰った夜、畑の野菜、固焼きパンの端、自家用に残していた肉で簡単なスープを作った。
- 翌朝、長期保存用の干し肉を作るため、脂の少ない肉を多めに欲しいと考え、ブランに次の狩りを頼んだ。
- 次の干し肉作りでは、流浪市で買った岩塩を使って塩加減と保存性を改善する方針。
- 二度目の鹿に似た獣を解体し、脂の少ない赤身を長期保存用の干し肉候補に、脂の多い部位を自家用に分けた。
- 皮、骨、脂、内臓も無駄にせず、皮は追加の袋や紐、骨は出汁や道具、脂は調理や保存、内臓は早めに加熱して自家用に回す候補とした。
- 二度目の獲物の内臓と脂の多い部位を、畑の野菜、塩、ハーブと一緒に煮込み、自家用料理にした。
- 二度目の獲物の皮を灰汁処理用に、骨を出汁用に、脂を溶かして保存するために分けた。
- 二度目の獲物の骨を煮出して骨出汁を取り、翌日までに使う前提で冷まして分けた。
- 二度目の獲物の脂を弱火で溶かし、布でこして小壺に入れた。
- 流浪市で買った岩塩を布に包んで石で砕き、粒の粗い塩として使えるようにした。
- 二度目の獲物の赤身を厚め、標準、薄めの三種類に切り分けた。
- 赤身の干し肉試作を、塩強め、標準、ハーブ強めの三種類に分けて仕込んだ。
- 試作した干し肉候補をあとで見分けられるよう、紐の結び方を変えて目印にした。
- 作業前の朝食として、前日に取った骨出汁、畑の野菜、固焼きパンの端を使ったスープを作った。
- 既存の簡易干し場に枝と紐を足して段を増やし、虫よけの布が肉に触れないよう外側に張り直した。
- 厚さと味付けが混ざらないよう、紐の結び方と吊るす位置で試作肉を分類して干した。
- 試作肉の途中経過を確認し、薄めの肉は乾きが早いが端が反りやすく、厚めの肉は中心に水分が残りやすいと観察した。
- 塩強めの試作肉は表面の締まりが早く、ハーブ強めの試作肉は香りがよい一方で乾き具合は標準と大きく変わらないと観察した。
- 標準の厚さと標準の塩加減が、売り物として扱いやすそうだと仮判断した。
- 干し肉の保存期間を試す間、市で売るものとして固焼きパンを追加で作ることにした。
- 粗い穀粉、水、塩、畑のハーブを使い、前回より形と厚みをそろえて固焼きパンを追加で焼いた。
- 市で売る候補として、固焼きパン単品よりも、固焼きパン、少量の干し肉、砕いた岩塩やハーブを組み合わせた旅人向け保存食セットを検討した。
- 追加で焼いた固焼きパンを完全に冷ましてから、布包み、木箱、皮袋に分けて保存性を試すことにした。
- 固焼きパンの保存試験で、布包みは短期用、木箱は形を守る用、皮袋は持ち運び用として比較する方針を立てた。
- 干し肉も固焼きパンも保存期間チェックに時間がかかるため、今ある銀貨でしばらく生活する選択肢を考えた。
- 肉はブランが狩ってくれるため、銀貨をすぐ増やそうと急がなくてもよい可能性を考えた。
- 流浪市で果物を買い、ジャムを作る案も考えたが、まだ実行していない。
- 紙の記録がないため、紐の結び方、木片の刻み目、置き場所で干し肉と固焼きパンの保存経過を追う仕組みを作った。
- 干し肉の試作は、厚さを紐の結び方、味付けを吊るす段、確認日数を木片の刻み目で管理することにした。
- 固焼きパンの保存試験は、布包み、木箱、皮袋の置き場所を固定し、それぞれの前に刻み目つきの木片を置いて管理することにした。
- 前世で「なろう系小説の世界の道具を使って、料理・保存食をつくる」YouTube動画を撮影していた経験から、低技術環境での干し肉と固焼きパンの保存期間をおおよそ推測できる。
- 脂の少ない赤身を十分に塩漬けし、乾燥と燻煙を徹底できれば、干し肉は少なくとも数週間以上の保存を目標にできると判断している。ただし、この世界の湿気、虫、夜露、保存容器に合わせて実測し、売る際の保証期間を決める方針。
- 固焼きパンは水分を抜いて乾燥を保てれば干し肉より長期保存に向き、数週間から数か月単位を目標にできると判断している。布包み、木箱、皮袋の試験は、長期保存の否定ではなく、この世界での最適な保管方法を選ぶためのもの。
- 前世の異世界想定保存食動画で得た知識を土台にし、この世界の湿気、虫、夜露、保存容器に合わせて調整する方針。
- 保存食の経過確認では、表面だけ乾いて中心に水分が残る状態、酸っぱい匂い、ぬめり、白・青・黒のカビ、湿り戻り、割った時の音や断面の違いを確認ポイントにしている。
- 干し肉と固焼きパンの一部を売り物にせず、割る、折る、匂いをかぐ、断面を見るための確認用サンプルとして扱う方針を立てた。
- 保存食試験の記録道具として、王都で小さなノート、インクの小瓶、葦ペンを購入した。
- 当面の自分の食糧計画として、畑の野菜と流浪市で買える安い傷あり野菜を使い、自家用の乾燥野菜を作る方針を立てた。
- 乾燥野菜は売り物ではなく、まず自分の食事とスープ用の備蓄にする方針。
- 畑の野菜で乾燥野菜の試験を始め、根菜は薄切り、葉物は重ねずに広げ、茎は細く裂き、香りの強い葉は少量ずつ分けて干す方針を確認した。
- 流浪市で傷あり野菜の籠を銅貨25枚で買い、自家用乾燥野菜の材料にした。
- 畑を、毎日食べる野菜、乾燥野菜に回す野菜、種取り用に残す株、香味草、秋まで様子を見る作物に分けて管理する方針を立てた。
- 畑の作物名がまだわからないため、見た目と用途で「根菜」「葉物」「香味草」「豆らしき莢」「秋まで待つ株」と分類し、ノートに畑の簡単な見取り図を描いた。
- 正規の市の野菜屋で畑の作物名を確認するため、身なりを整えて王都へ向かい、ブランを普通の犬の姿で王都の門脇に待機させた。
- 正規の市の野菜屋で、畑の根菜がルタ、細長い橙色の根菜がカロ、葉物がラウナ、香味草がセリム草、豆のような莢が莢豆、秋まで待つ株が土芋の可能性が高いと教わった。
- 正規の市の野菜屋で、ルタ2つ、カロ2本、ラウナ少量を銅貨18枚で買った。
- ノートに「畑管理表」を作り、ルタ、カロ、ラウナ、セリム草、莢豆、土芋を、食用、乾燥用、種取り用、観察用に分けて記録し始めた。
- 骨を畑の肥料に使える可能性を思い出し、煮出した後の骨を乾かして焼き、砕いて少量ずつ土に混ぜる方針を立てた。
- 乾かした煮出し後の骨をかまどの余熱で焼き、石で砕いて粗い骨粉にし、土芋、ルタ、カロの試験株へ少量だけ混ぜた。
- 二度目の獲物の皮を確認し、大きな一枚革として使うより、皮紐、荷紐、袋の補強、岩塩やハーブ用の小袋、ノートとインクを守る包み、干し場の吊り具に分けて活用する方針を立てた。
- 皮は食品に直接触れさせる用途には慎重で、干し肉や固焼きパンは布や木箱を優先し、皮袋は持ち運び用や外側の保護に使う方針。
- 現在の所持金を確認し、手元には銀貨8枚と銅貨22枚が残っていると把握した。
- 運搬量を増やすため、ブランに引かせられる犬用の二輪荷車を王都の木工工房で作ってもらうことにした。小型ではあるが飾りや軽荷用ではなく、岩塩や野菜籠、固焼きパンの木箱など重い荷も運べる作業犬用の頑丈な荷車。
- 荷車本体の価格は銀貨3枚と銅貨50枚。手付金として銀貨1枚を払い、残金は完成時に銀貨2枚と銅貨50枚。
- 荷車の引き具は工房任せにせず、二度目の獲物の皮でブランの体に合わせた幅広の革紐を自作する方針。
- 荷車の手付金を払った後の所持金は銀貨7枚と銅貨22枚。
- 家に帰ってから、ブラン用荷車の引き具制作に入った。首輪で引かせず、胸当て、肩紐、胴回りの支え、引き棒につなぐ左右の革紐で力を受ける構造にする方針。
- ブランの普通の犬の姿に合わせて採寸し、大きな狼の姿にも流用できるよう、穴と結び目で長さを調整できる仮作りにした。
- 翌朝、ブランの胸、肩、脇、首回りに擦れ跡がないことを確認し、普通の犬の姿での仮引き具は大きな問題がないと判断した。
- ブランの大きな狼の姿でも仮合わせし、胸当てと肩紐は使えるが、胴回りの支えと左右の引き紐は延長が必要だと判断した。
- 引き具を、普通の犬姿用の穴と大きな狼姿用の穴を分け、森では延長用の革紐を足して使う二段階調整式にする方針を立てた。
- ブラン用引き具の胸当ての裏に柔らかい革を重ね、脇に当たりそうな角を丸く切り、肩紐と胴回りの革に普通の犬姿用と大きな狼姿用の穴を分けて開けた。
- 荷車完成前の仮負荷試験として、引き具の左右紐に軽い薪束をつなぎ、ブランに前進、停止、ゆっくり曲がる動きを試してもらった。
- 翌日、ブラン用引き具を完成させた。肩紐の下に柔らかい革を重ね、胸当てと胴回りの支えを本縫いし、普通の犬姿用と大きな狼姿用の調整穴、延長用の革紐、左右の引き紐を仕上げた。
- 完成した引き具は、首輪ではなく胸と肩で力を受け、胴回りでずれを抑える構造。荷車の引き棒の高さに合わせて左右の引き紐の結び位置を最終調整する余地を残している。
- 荷車を受け取る日、残金の銀貨2枚と銅貨50枚を持って、ブランとともに王都の木工工房へ向かった。
- 木工職人に残金を支払い、作業犬用の頑丈な小型二輪荷車を受け取った。支払い後の手元の所持金は銀貨4枚と銅貨72枚。
- 工房でブランに引き具を装着し、木工職人に確認してもらいながら荷車の引き棒と左右の引き紐の位置を調整した。
- 干し肉と固焼きパンの保存試験を、仕込み日を0日目として数え、荷車受け取り後の確認時点を12日目として記録した。
- 12日目の確認で、干し肉に酸っぱい匂い、ぬめり、白・青・黒のカビがなく、標準厚・標準塩のものは表面と中心の乾き具合が安定していると判断した。
- 12日目の確認で、固焼きパンに湿り戻り、カビ、嫌な匂いがなく、叩いた音も乾いており、布包みと底上げした木箱の保存状態が良好だと判断した。
- 12日目の確認後、固焼きパン2枚、干し肉1枚、砕いた岩塩少量、乾燥セリム草少量を布包みにした旅人向け保存食セットを考え、銅貨60枚前後を基準価格にする方針を立てた。
- 販売時に「十二日確認済みで異常なし」と説明するが、まだ数か月保存を保証するとは言わない方針。
- 12日持った試験品は検証用として残し、同じレシピで販売用の干し肉と固焼きパンを新たに作る方針を立てた。
- 販売用の新しい干し肉を仕込むため、ブランに翌日の狩りを頼んだ。
- 翌日、ブランが狩った鹿に似た獣を使い、試験品とは別に販売用の干し肉を仕込み始めた。条件は12日目に異常がなかった標準厚・標準塩・軽い燻煙にそろえる方針。
- その翌日、12日目に異常がなかった固焼きパンと同じレシピ、厚さ、焼き方で、販売用の固焼きパンを新たに作った。試験品は継続観察用として残す方針。
- その次の日、干し肉と固焼きパンの乾燥を待つ空き時間に、なめした皮を加工し、保存食セットの外側を守る小さな革包み、岩塩や乾燥セリム草用の小袋、荷車や木箱に固定するための革紐を作り始めた。食品に直接触れる部分は布を使い、革は外側の保護と固定用にする方針。
- 数日後、販売用干し肉を確認し、標準厚・標準塩のものについて、酸っぱい匂い、ぬめり、白・青・黒のカビがなく、割った断面にも湿った芯が残っていないため完成と判断した。試験品の長期観察は継続する方針。
- その翌日、完成した販売用干し肉、販売用固焼きパン、砕いた岩塩、乾燥セリム草を使い、旅人向け保存食セットを布包みにして組み始めた。初回販売価格は1包み銅貨60枚に決め、試食用の小片と説明用の確認サンプルを別に用意し、翌日流浪市へ少量持っていく方針を立てた。
- 翌日、ブランを普通の犬の姿にし、完成した保存食セット、試食用の小片、説明用サンプルを木箱に入れて荷車で王都へ向かった。門番には保存食を売りに来たと説明し、流浪市の受付で再び敷物と場所を借りて店を広げた。
- 流浪市でアレックスに保存食セットを試食してもらい、旅人用の一日分として銅貨60枚は妥当だと評価された。アレックスは保存食セットを2包み購入し、代金は銅貨120枚になった。支払い後の手元は銀貨5枚と銅貨92枚相当。
- アレックスから、保存食セットだけでなく、干し肉だけ欲しい客もいるはずだからバラ売りもしたほうがよいと助言された。
- アレックスの助言を受け、保存食セットとは別に干し肉単品を1枚銅貨30枚で並べ始めた。試食した旅人が干し肉単品を2枚購入し、代金は銅貨60枚になった。支払い後の手元は銀貨6枚と銅貨52枚相当。
- 保存食セット初回販売で、最初は敷物に少量だけ並べたが、売れ行きを見て荷車の木箱から追加し、最終的に保存食セット12包み、干し肉単品30枚、固焼きパン単品12枚を完売した。保存食セットは1包み銅貨60枚、干し肉単品は1枚銅貨30枚、固焼きパン単品は1枚銅貨15枚で販売した。売上は合計銅貨1800枚で、完売後の手元は銀貨22枚と銅貨72枚相当になった。
- 完売後、ジェイクの店へ行き、岩塩を2袋購入した。岩塩は1袋銀貨1枚のため、支払いは銀貨2枚。購入後の手元は銀貨20枚と銅貨72枚相当。
- 家に帰ると、岩塩2袋を貯蔵庫へ入れ、銀貨を小さなツボにしまい、現在の所持金が銀貨20枚と銅貨72枚であることを確認した。
- その翌日、売らずに残している保管実験中の干し肉と固焼きパンを確認し、干し肉に酸っぱい匂い、ぬめり、白・青・黒のカビはなく、固焼きパンにも湿り戻り、カビ、嫌な匂いがないことを確認した。長期観察は継続する方針。
- 保管実験中の干し肉と固焼きパンの確認日を、仕込み開始から21日目、12日目の確認から9日後としてノートに整理した。
- 保存実験の目標として、標準レシピの干し肉は30日を次の節目、60日前後を実用限界候補、固焼きパンは90日を中間節目、木箱保管で180日前後を実験上限候補に設定した。ただし、商売で保証するのは実測済みの日数だけにする方針。

- 王都で買った粗い穀粉25kg袋について、すぐ使う分だけ布袋へ小分けし、残りは大袋の口を折って紐で縛り、布で包んで木台の上に置く保管方針を立てた。
- 穀粉の小分け袋は、固焼きパン十個分を目安にした。量りがないため、同じ木椀で同じ回数すくって袋ごとの差を減らし、あとで試し焼きして分量を調整する方針。
- 穀粉の虫・湿気対策として、食品に直接触れる部分は布、外側の固定は革紐、保管は底上げした木箱と木台を使うことにした。粉袋は床や壁に直接触れさせず、空気が少し通るように置く。
- 乾燥セリム草を虫よけ候補として使うが、粉に匂いが移りすぎないよう、直接混ぜず、布で包んだ小袋を粉袋に触れない位置へ置いて比較する方針。
- 穀粉の確認を三日後、七日後、十五日後に行い、湿気による固まり、虫の痕跡、匂い移りを確認する方針を立てた。
- 小分けした穀粉袋を一つ使い、固焼きパンの試し焼きを行った。木椀十杯分の穀粉で、販売用の標準サイズ十枚と試食用の小片を作れると確認した。
- 販売用固焼きパンの成形について、厚すぎると中心に湿りが残り、薄すぎると割れやすいため、手のひらより少し小さい平たい円形を標準にする方針を立てた。
- 固焼きパンを一度焼いたあと、火から外したかまどの余熱でさらに乾かし、割った時に鈍い湿った音ではなく乾いた音が出ることを完成目安にした。
- 試し焼き後、小分け袋一つを「固焼きパン十枚分」として管理し、販売用の仕込み計画を袋単位で立てる方針を決めた。
- 干し肉を乾かしている間に固焼きパンを多く作れるため、保存期間に影響が出ない範囲で味違いの試作品を作る方針を立てた。ただし販売の本線は標準レシピのままにし、味違いは少量試験から始める。
- 固焼きパンの味違い試作では、水分や脂を増やす材料を避け、乾燥セリム草、砕いた岩塩など乾いた材料だけを候補にする方針。生野菜、脂、甘い煮詰め物は保存期間に影響しやすいため、当面は入れない。
- 固焼きパンの味違い試作として、標準、乾燥セリム草少量入り、乾燥セリム草やや多め、砕いた岩塩少量入りを小さく分けて作り、標準品とは別に保存経過を観察する方針を立てた。
